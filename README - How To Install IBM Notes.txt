Hello, Thank you for downloading IBM Notes 9.0.1 for Ubuntu 12.04

In order for this to work completely please copy all downloaded files to your Desktop. 

Once you have done that. 

Please right click on "IBM Notes 9.0.1 Installer.sh" and select properties.


Select the tab "Permissions" 


Please check the box 
Execute: Allow Executing file as program.


Click Close



Now double click on the Installer, and Click "Run in Terminal"
